Objects

Every thing you'll deal with belongs to some type or another. We say that it is an instance of that type.

14 is an instance of Number, "Hello, World!" is an instance of String

There is one particularly useful type of thing called Object

Instances of Object can hold on to other things. It's like a variable that holds a bunch of other variables. These are called properties.

We get to an object's properties by using a dot (.) operator.

This page has an object named teach. In the console, see what's in the teach object's name property using the dot operator, like this:
teach.name

Just like we used quote marks to define a thing of type String, we can use some other special characters to define a thing of type Object. When we do that, we're dealing with literals. Numbers have the easiest literals because they are just numbers like you're used to. Strings make sense, because you've often seen words inside of quotes. The Object literal is a little more tricky, and it'll just take some time to get used to. Type this in your console.

var person = {firstName:"Sam", lastName:"Spade", age:24};

Now type
person.firstName;

Now try to add new properties using the dot:
person.bestFriend = "me";
