The Console

You're going to learn to program by hacking.

Don't worry, it's legal. Maybe you're disappointed about that.

Hacking, in this case, just means getting inside a system to learn how it works and then make it do things you want it to.

As it happens, you are looking at the world's most widely used programming platform: The Web Browser. The venerable web browser these days also happens to be one of the most hackable platforms ever, thanks to the console.

When you do what I'm about to tell you to do, a window may appear. If it does, move it and resize things so you can see this window again. If you're using Firefox or Chrome, right-click anywhere on this page and select "Inspect Element".

Look for something called "console" in the new window.
